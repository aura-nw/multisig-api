# Multisig-api
Multisignature backend api 

## Design architecture
Click [here](docs/README.md) to see the design architecture.
## Installation

```bash
$ npm install
```

## Create migrations
```bash
$ typeorm migration:create -n <name migrations>
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
